# Superfier

AI chat application built with AI-SDK.

## Features

- Chat with AI-powered agents
- Save and manage chat history
- Responsive design for all devices
- Customizable theme

## Tech Stack

- Next.js (app router)
- Tailwind CSS / Shadcn UI
- Clerk / Auth
- Prisma / Postgres
- AI SDK / OpenAI / Anthropic
